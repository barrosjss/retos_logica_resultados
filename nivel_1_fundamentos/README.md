# ⭐ Nivel 1 — Fundamentos del Pensamiento Algorítmico

> **Conceptos:** Secuencias, variables, entrada de datos, salida de resultados, operaciones básicas

---

## ¿Qué aprenderás en este nivel?

El pensamiento algorítmico es la capacidad de descomponer un problema en pasos ordenados, precisos y sin ambigüedad. En este nivel practicarás:

- 📥 **Entrada de datos**: recibir información del usuario
- 📤 **Salida de resultados**: mostrar información procesada
- 🔢 **Variables**: almacenar valores temporales
- ➕ **Operaciones básicas**: suma, resta, multiplicación, división, módulo
- 📋 **Secuencias**: instrucciones en orden específico

## 💡 Consejos para este Nivel

- **No compliques lo simple**: los problemas de este nivel parecen simples, pero la clave está en ser **preciso**
- **Identifica las variables** antes de empezar el diagrama: ¿qué datos necesitas guardar?
- **Un rectángulo por operación**: no hagas un rectángulo con 5 cosas al mismo tiempo

---

## 📐 Símbolo más usado en este nivel

```
[/Leer dato/]        ← Paralelogramo para entrada
[resultado = a + b]  ← Rectángulo para proceso
[/Mostrar resultado/] ← Paralelogramo para salida
```
