# ⭐⭐ Nivel 2 — Condicionales y Decisiones

> **Conceptos:** If/Else, comparaciones, operadores lógicos (AND, OR, NOT), decisiones anidadas

---

## ¿Qué aprenderás en este nivel?

El mundo real está lleno de condiciones: "si el semáforo está en verde, avanza; si está en rojo, detente". Los programas también toman decisiones. En este nivel practicarás:

- 🚦 **If / Else**: tomar un camino u otro según una condición
- 🔗 **Operadores lógicos**: combinar condiciones con AND, OR, NOT
- 🪆 **Condicionales anidados**: decisiones dentro de decisiones
- 📊 **Comparaciones**: igual, diferente, mayor, menor, mayor-igual, menor-igual

## 💡 Consejos para este Nivel

- **El rombo siempre tiene DOS salidas**: Sí y No (nunca tres)
- **Cuidado con los condicionales anidados**: dibújalos ordenados para no perderte
- **Escribe la condición completa en el rombo**: no solo "edad", sino "¿edad >= 18?"
- **Prueba los casos límite**: ¿qué pasa exactamente cuando el valor es igual al umbral?
